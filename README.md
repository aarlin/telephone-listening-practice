# Phone Number Listening Practice

A language learning tool that helps you practice listening comprehension by transcribing spoken phone numbers in different languages. This game is perfect for improving your number recognition and listening skills in various languages.

## 🎯 Purpose

When learning a new language, understanding numbers in spoken form can be challenging. This tool helps you practice by:
- Listening to phone numbers spoken in your target language
- Transcribing what you hear
- Getting immediate feedback on your accuracy
- Practicing at different speeds and difficulty levels

## 🌟 Features

- **Multiple Languages**: Practice with numbers spoken in:
  - English
  - Spanish
  - French
  - German
  - Italian
  - Mandarin

- **Adjustable Difficulty**:
  - Easy: Normal speaking speed
  - Medium: 1.5x faster
  - Hard: 2x faster

- **Flexible Number Lengths**:
  - 7 digits (standard phone numbers)
  - 10 digits (international format)
  - 11 digits (some country formats)

- **Real-time Feedback**:
  - Immediate scoring
  - Visual feedback for correct/incorrect answers
  - Continuous practice mode

## 🚀 Getting Started

1. **Prerequisites**:
   - Node.js (v14 or higher)
   - npm (v6 or higher)

2. **Installation**:
   ```bash
   # Clone the repository
   git clone https://github.com/yourusername/telephone-listening-practice.git
   cd telephone-listening-practice

   # Install dependencies
   npm install
   ```

3. **Running the Application**:
   ```bash
   npm run dev
   ```

4. **Using the Application**:
   - Open your browser to `http://localhost:5173`
   - Select your target language
   - Choose your preferred difficulty level
   - Select the number length you want to practice
   - Click "Start New Number"
   - Type the numbers as you hear them
   - Press Enter to check your answer

## 🎮 How to Play

1. Select your settings:
   - Choose the language you want to practice
   - Set your preferred difficulty level
   - Pick the number length you want to practice

2. Start the game:
   - Click "Start New Number"
   - Listen carefully to the spoken numbers
   - Type the numbers as you hear them
   - You can type while the numbers are being spoken

3. Check your answer:
   - Press Enter when you're done typing
   - Get immediate feedback on whether you were correct
   - Your score will update accordingly

4. Continue practicing:
   - Click "Start New Number" to try another number
   - Keep practicing to improve your listening skills

## 💡 Tips for Success

- Start with the "Easy" difficulty to get accustomed to the numbers
- Practice regularly to improve your recognition speed
- Try different languages to challenge yourself
- Use headphones for better audio quality
- Focus on the rhythm and pattern of the numbers
